Mengenal Semantic Elements di HTML5

Article

Ketika kita ingin membuat layout untuk menamplikan block artikel-artikel yang ada di website kita, tag artikel bisa kita gunakan dengan elements header dan footer.Karena tag article memiliki type block, artinya element ini memiliki ebar sesuai lebar browsr jika kita tidak menetukan lebarnya.

Aside

Element ini bisa digunakan untuk melengkapi layout utama.Misal kita memiliki halaman blog maka kita bisa bisa menambahkan asidedi bagian bawah postingan mu karena aside memiliki type/sifat block maka lebarnya sesuai page jika kita menentukan lebarnya.
